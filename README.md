### Quick start

Use the following command to quickly start Dr.Dentist project.

```bash
git https://github.com/phamquangvinhfpt/dr-dentist-vuestic.git
cd DR-DENTIST-VUESTIC
npm install
npm run dev
```

### How to started mobile app

- Run build project:

```bash
npm run build
```

- Sync project:

```bash
npx cap sync
```

- Open android studio

```bash
npx cap open android
```
